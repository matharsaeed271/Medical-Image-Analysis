# Medical-Image-Analysis
Built an AI-powered Medical Image Analysis Tool that helps users understand medical images by generating detailed AI-assisted reports. The application allows users to upload an image, analyzes it using Google's Gemini 2.5 Flash multimodal model through the Agno framework, and provides clear, context-aware explanations in an easy-to-understand format.

To improve the quality of the responses, the application also performs web-assisted research and combines relevant medical information with AI-generated insights. This project gave me hands-on experience in developing multimodal AI applications, integrating AI agents, prompt engineering, and building interactive web applications.

# Key Features
Medical image analysis using Gemini 2.5 Flash
AI-generated reports with contextual explanations
Web-assisted research for enhanced responses
Interactive chat interface for follow-up questions
User-friendly Streamlit web application
AI agent workflow powered by the Agno framework
